# Zoo DirectoryDeveloped by Xiaoyuan Lai, in May 2016 for the class COEN268 project 2.This project provides a listing of animals, detailed information of each animal, and basic information about the zoo.Image files stored in /assets. This app does not use separate image for icon and full-view.This app support both portrait and landscape modes.

project special setting:
set /build.gradle(Module: app) targetSdkVersion to 22

Download to local file systemUse Android Studio to open the project.
